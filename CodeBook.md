## Code Book

This is a code book for final export `tidy.txt`.

### Subject and Activity

- `subject` - integer specifying the ID of a subject
  - values from 1 to 30
- `activity` - activity of a subject during the measuremenet
  - `WALKING`
  - `WALKING_UPSTAIRS`
  - `WALKING_DOWNSTAIRS`
  - `SITTING`
  - `STANDING`
  - `LAYING`
